Conclusion

Here we can see that when we use an Unordered List, the bullet points created can have any shape, which can be changed according to our convenience. On the other hand, when we use an Ordered List, it uses lexicographically ordered characters to represent each value inside the "li" tag, just like in shapes. Additionally, we can also change the style of the ordered list.
